힌트 About

왜 만들었나

차즘의 "숨겨진 차키를 찾아라" 이벤트 힌트를 한 화면에 정리해 보기 쉽게 만든 비공식 큐레이션 페이지입니다. 공식 채팅방에서는 힌트가 빠르게 흘러가기 때문에, 뒤늦게 합류했거나 흐름을 놓친 사람이 전체를 한눈에 훑어볼 수 있도록 카드 뷰로 정제했습니다.

운영 방침

  • 공식 채팅방에 공개된 힌트만 수록 · 사전 유출은 다루지 않음
  • 추정 위치는 커뮤니티 공감대가 있을 때만 기재
  • 이미지 저작권 확인 전까지 그림카드 업로드는 보류
  • 업데이트는 수동 · 하루 1~2회 반영

기술 스택

  • 정적 HTML + Alpine.js (가벼운 반응형 UI)
  • 단일 hints.json 데이터 파일 (GitHub repo 관리)
  • Cloudflare Pages 배포 · CDN 엣지 캐시
  • 운영 비용 0원 · 트래픽 스파이크 대응

공식 링크

  • 차즘 공식 이벤트 페이지
